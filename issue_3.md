# Issue 3: UIの改善と可視化機能の実装

## 概要
Next.jsとTailwind CSSを使用し、医療用アプリケーションとして信頼性が高く、使いやすいモダンなユーザーインターフェースを構築する。

## 詳細タスク

### 1. プロジェクトセットアップ
- [ ] `web/` ディレクトリで `npx create-next-app@latest` を実行（TypeScript, Tailwind CSS有効）。
- [ ] 基本的なレイアウト（ヘッダー、メインコンテンツ、フッター）を作成する。

### 2. 画像アップロードUI
- [ ] ドラッグ＆ドロップ対応のアップロードエリアを実装する（`react-dropzone` 等の使用を検討）。
- [ ] アップロード中のローディング表示（スピナーやプログレスバー）を実装する。

### 3. 推論結果の可視化
- [ ] **画像表示**: アップロードした元画像と、APIから返却されたGrad-CAM画像（ヒートマップ）を並べて（または切り替え可能にして）表示する。
- [ ] **確率グラフ**: 各クラスの予測確率を棒グラフや円グラフで視覚的にわかりやすく表示する（`recharts` や `chart.js` 等の使用を検討）。
- [ ] **診断サポート**: 最も高い確率の病変名をハイライト表示する。

### 4. デザイン調整
- [ ] 配色: 医療系に適した清潔感のある配色（青、白、グレー基調）を採用する。
- [ ] レスポンシブ対応: デスクトップだけでなく、タブレットサイズでも崩れずに表示できるようにする。

## 完了条件
- ユーザーが直感的に画像をアップロードできること。
- 推論結果が視覚的にわかりやすく表示されること。
- 全体として統一感のあるプロフェッショナルなデザインであること。
